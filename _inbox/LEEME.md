# _inbox — Zona de entrada

Sube aquí tu curaduría de información de las carpetas viejas. Para que Claude la clasifique bien, junto a cada archivo (o en un `.txt` al lado) indica:

1. **Marca:** alphatype / (otra)
2. **Qué es:** identidad / diseño / voz / producto / contenido / ecommerce / crm / data / social
3. **De dónde salió:** ej. "venía de Alphatype-Brand-Book/04_Voice"
4. **Nota opcional:** "esta es la versión buena, ignora la de Design-System"

No tienes que ser perfecto. Si no sabes dónde va algo, súbelo igual y Claude pregunta o lo deduce.

Cuando termines de subir, dile a Claude: **"procesa el inbox"** y él clasifica todo, anota el origen en `_migration/`, y deja esta carpeta vacía.
